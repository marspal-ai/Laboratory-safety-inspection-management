# Laboratory-safety-inspection-management实验室安全检查（安全巡检）管理

#### 系统介绍

重点打造实验室安全的闭环化管理，从隐患识别上报、隐患评估分类、隐患整改措施实施、整改效果的跟踪与反馈直至最后的持续改进与优化。内容包括:发布检查任务、执行检査任务、提交检査结果、记录安全隐患、整改安全隐患,核实整改结果，安全隐患消除。即时通知:当系统接收到新的隐患报告时，立即通过短信、推送等方式通知相关责任人(如实验室管理员、安全负责人等)超期提醒:如果隐患未在规定时间内得到响应或整改，系统会自动发送超期提醒。

#### English version introduction

#### System Introduction

Introduction to Laboratory Safety Inspection System

We will focus on building a closed-loop management system for laboratory safety, including hazard identification and reporting, hazard assessment and classification, implementation of hazard rectification measures, tracking and feedback of rectification effects, and continuous improvement and optimization in the end. The content includes: publishing inspection tasks, executing inspection tasks, submitting inspection results, recording safety hazards, rectifying safety hazards, verifying rectification results, and eliminating safety hazards. Instant notification: When the system receives a new hazard report, it immediately notifies relevant responsible persons (such as laboratory administrators, safety managers, etc.) through SMS, push notifications, etc. Overdue reminder: If the hazard is not responded to or rectified within the specified time, the system will automatically send an overdue reminder.
